# App Store Content 审核状态

App Store Connect(iPhone) (opens new window) 掌上了解App在AppStore的状态。

2018年06月07日,苹果今天正式宣布，开发者工具系统 iTunes Connect(iPhone) (opens new window) 已变更为 App Store Connect(iPhone) (opens new window)

help.apple.com/app-store-connect (App Status) (opens new window) 应用程序状态立即表明您的应用程序是否需要注意。

# 1. 说明

  • 红色状态指示器 (🔴 )意味着您需要执行一些操作, 然后应用程序才可以在应用程序商店中使用。
  • 黄色状态指示器 (💛 )意味着某些过程正在进行, 由您或苹果控制。
  • 绿色状态指示器 (💚 )意味着该应用程序已准备好出售。

# 2. 状态

  • 💛 Prepare for Submission 准备提交
  • 💛 Waiting For Review 等待审阅
  • 💛 In Review 在审查中
  • 💛 Pending Contract 待定合同
  • 💛 Waiting For Export Compliance 正在等待导出符合性
  • 💛 Pending Developer Release 正在等待开发人员发布
  • 💛 Processing for App Store 应用程序存储的处理
  • 💛 Pending Apple Release 搁置苹果发行版
  • 💛 Pending App Transfer 挂起的应用程序传输
  • 💛 Processing App Transfer 处理应用程序传输
  • 💚 Ready For Sale 准备出售(App 上线，如果是手动发布的需要开启操作)
  • 💚 Pre-Order Ready for Sale 订货前准备出售
  • 🔴 Rejected 拒绝
  • 🔴 Metadata Rejected 拒绝元数据
  • 🔴 Removed From Sale 从销售中删除
  • 🔴 Developer Rejected 开发人员拒绝
  • 🔴 Developer Removed From Sale 开发人员从销售中删除
  • 🔴 Invalid Binary
