# 简介

## 一. iPhone - App
1. `Developer` : Apple讲解视频、活动信息
1. `Connect` : ipa 上传到 App Store Connect 成功后有推送通知，也可以查看app的审核状态。
1. `TestFlight` : 测试app的软件
1. `AirPort 工具` : iPhone 之间传输文件
